综述

本类是Java集合框架中的一员。提供了Set接口的最小化实现。E表示被Set所持有的元素类型。

在遵循Set接口的额外约束(例如不允许存在重复元素)的前提下，继承本类实现Set与继承AbstractCollection实现Collection所需做的工作是相同的。
